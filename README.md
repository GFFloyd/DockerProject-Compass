# Atividade de Docker na Cloud AWS
_Estágio DevSecOps | Compass.UOL_

---

## Introdução

A atividade pede para criarmos instâncias EC2 com Wordpress conteinerizado via Docker, estas instâncias devem se conectar com o RDS (para o Wordpress conectar ao MySQL) da Amazon e ter conexão com o EFS (para guardar as informações estáticas do site).
Estas instâncias devem ser geridas pelo Load Balancer e criadas automaticamente pelo Auto-Scalling.
